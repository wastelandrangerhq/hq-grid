CGW's November 1996 CG-ROM Wasteland installation tips

Here are some tips on installing Wasteland from the Computer Gaming World November 1996 CG-ROM:

Windows 95:

  1. Copy the whole \demos\interply\wl directory to your hard drive (or a floppy)
  2. In Windows 95 or Windows NT, select all the files in the wl directory
  3. Select File: Properties
  4. Uncheck Read-Only
  5. Run wl.exe
DOS:
  1. Copy the whole \demos\interply\wl directory to your hard drive (or a floppy)
  2. Switch to the wl directory
  3. Execute the command: attrib -r *.*
  4. Execute wl.exe
